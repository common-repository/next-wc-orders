Next Orders for WooCommerce helps you easily view and filter your shop orders and export them !

== Description ==
Next Orders for WooCommerce is the best orders manager for Woo Commerce. It automatically displays the orders you are looking for, according to different filters (Free shipping, year, status, total, product name, product category...).
It generates stats with pie and bar charts of number of orders, sales per month or year, product per category...
It lets you export your result in several formats (CSV or HTML).

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/next-wc-orders` directory, or install the plugin through the WordPress plugin screen directly.
3. Activate the plugin through the 'Plugins' screen in WordPress
2. Go to Next Orders menu to configure the plugin and save your changes.
